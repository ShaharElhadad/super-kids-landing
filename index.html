// QUEST SYSTEM

כל אימון הוא משימה

אפליקציה שמרגישה כמו משחק. כל פעילות נותנת XP, כל יום מביא אתגר חדש, וכל שבוע מתקרבים לדרגה הבאה.

אתגר יומי

כל בוקר מחכה אתגר חדש ומותאם אישית. מ־20 סקוואטים ועד ריצה של 10 דקות. תמיד משהו חדש.

בנה אימון משלך

תבחר תרגילים, תגדיר חזרות וזמנים, ותבנה אימון מותאם אישית. או תבחר מאימונים מוכנים מהספרייה.

מערכת XP ודרגות

כל פעילות שווה נקודות ניסיון. צבור XP, עלה בדרגה, ופתח הישגים מיוחדים. מברונזה ועד אגדה.

תחרות עם חברים

הזמן חברים, התחרה בלידרבורד שבועי, ושלח אתגרים 1 נגד 1. מי יגיע ראשון לדרגה הבאה?

מעקב התקדמות

גרפים ודשבורדים שמראים את ההתקדמות לאורך זמן. סטריק יומי, סטטיסטיקות שבועיות, ותובנות חכמות.

תזונה בריאה

מעקב אוכל בריא, מתכונים לילדים, ואתגרי תזונה שבועיים. לאכול נכון זה חלק מהמשחק.

// RANK SYSTEM

עלה בדרגה. הפוך לאגדה.

כמו ב־Solo Leveling, כל שחקן מתחיל מלמטה ועולה. 6 דרגות, כל אחת עם יכולות ואתגרים חדשים.

E
E RANK — מתחיל
צעדים ראשונים במסע הכושר. תרגילים בסיסיים, קצב נוח, והמון עידוד.
0 — 500 XP
D
D RANK — מתאמן

500 — 2,000 XP
C
C RANK — לוחם
כבר מרגישים את השינוי. אימונים אינטנסיביים וגישה לאתגרי בוס שבועיים.
2,000 — 5,000 XP
B
B RANK — אלוף
ברמה גבוהה. אתגרי PvP נפתחים, לידרבורד ארצי, ואימונים מתקדמים.
5,000 — 15,000 XP
A
A RANK — מאסטר
עילית. יכולת ליצור אתגרים לאחרים, תכנים בלעדיים, ובאדג׳ מיוחד.
15,000 — 50,000 XP
S
S RANK — אגדה
הדרגה הגבוהה ביותר. רק האמיצים מגיעים לכאן. סטטוס אגדי לצמיתות.

// HOW TO PLAY

איך מתחילים?

4 צעדים פשוטים והמסע מתחיל

01

הורד את האפליקציה

זמין ב־App Store ו־Google Play. הרשמה תוך 30 שניות.

02

צור את הדמות שלך

בחר אווטאר, שם גיבור, והתחל כ־E Rank. המסע מתחיל.

03

קבל את האתגר הראשון

האתגר היומי מחכה לך. השלם אותו וקבל את ה־XP הראשון שלך.

04

עלה בדרגה

המשך להתאמן כל יום, צבור XP, ועלה בדרגות. היעד: S RANK.
